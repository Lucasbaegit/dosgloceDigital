# Implementación Autoadhesivas v1

## Estado operativo
- Papel: `AUTOADHESIVA_PAPEL_HIBRIDO_B_C`.
- Especial: `AUTOADHESIVA_ESPECIAL_HIBRIDO_B_C`.
- Fuente operativa para ambos: tabla objetivo PDF por rango.

## Trazabilidad técnica
- Papel mantiene origen Excel `Bajadas!S4:S10` (Sticker).
- Especial mantiene origen Excel `Bajadas!U4:U10` (OPP blco o blanco).
- `MODELO_B3` queda como referencia técnica para especial, no como precio operativo.
