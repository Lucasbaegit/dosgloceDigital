# Trazabilidad Autoadhesivas v1

## Papel
- Regla: `AUTOADHESIVA_PAPEL_HIBRIDO_B_C`
- Operativo: precio objetivo PDF exacto por rango.
- Referencia Excel: `=(AC4x + AC12*AF4x)*AH4x`

## Especial
- Regla: `AUTOADHESIVA_ESPECIAL_HIBRIDO_B_C`
- Operativo: precio objetivo PDF exacto por rango.
- Referencia Excel: `=(AC4x + AC13*AF4x)*AH4x`
- Modelo técnico referencia: `MODELO_B3`
- Motivo operativo: PDF vigente usado como precio final comercial.

## Fuera de alcance
- Tinta blanca
- Laca UV
- Adicionales / terminaciones
- Medio corte / precorte
